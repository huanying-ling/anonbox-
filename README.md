
  # Anonbox

  一个轻量级的匿名提问箱网页应用。用户可以创建自己的提问箱，接受匿名提问并公开回答；管理员拥有
  全站管理权限。

  ## 功能亮点

  - **创建提问箱** — 注册即拥有唯一子页面，支持自定义昵称、头像、URL标识
  - **匿名提问** — 访客无需登录即可匿名提问，支持可选昵称
  - **访问密码** — 提问箱可设置密码保护，主页访问需验证，直接 URL 可分享
  - **问题管理** — 公开/隐藏/回答/置顶/删除提问，字数限制、敏感词过滤
  - **管理员后台** — 全站数据概览、问题筛选、IP 封禁、提问箱管理、公告编辑
  - **一键部署** — 支持宝塔面板 + Nginx + PM2，详细部署文档
  - **响应式设计** — 适配手机、平板、桌面端，简洁现代 UI

  ## 技术栈

  Node.js + Express + SQLite（sql.js）+ EJS 模板引擎 + 原生 HTML/CSS/JS

  ## 快速开始

  \`\`\`bash
  cp .env.example .env
  npm install
  npm start
  \`\`\`

  访问 http://localhost:3000，默认管理员 admin / admin123
